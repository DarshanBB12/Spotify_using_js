~ Spotify Clone

A simple Spotify Clone Music Player built using HTML, CSS, and JavaScript.
This project replicates the basic layout and functionality of Spotify where users can play songs through a clean and interactive interface.

~Features

> Play and Pause songs
> Next and Previous song controls
> Song list display
> Responsive layout
> Clean Spotify-like user interface

~Technologies Used

>HTML5 – Structure of the web page
>CSS3 – Styling and layout design
>JavaScript – Music player functionality and DOM manipulation


~Learning Goals

This project helped me learn:
   >JavaScript DOM manipulation
   >Audio control using JavaScript
   >Building interactive web interfaces
   >Creating responsive layouts
